# Visualisation du PIB des États-Unis avec D3.js

Ce projet est une visualisation interactive réalisée avec D3.js représentant l’évolution du Produit Intérieur Brut (PIB) des États-Unis, trimestre par trimestre, de 1947 à 2015. Les données proviennent du dépôt officiel de FreeCodeCamp et ce projet respecte l’ensemble des user stories du projet “Bar Chart”.

## Présentation générale

Le graphique affiche l’évolution du PIB sous forme de barres verticales. Chaque donnée correspond à un trimestre (Q1, Q2, Q3, Q4). Une info-bulle dynamique apparaît au survol et indique :
- l’année et le trimestre (ex : 1967 Q3)
- la valeur du PIB en milliards de dollars

Les axes sont générés automatiquement via D3.js, tout comme les échelles temporelles et linéaires. Un label vertical est ajouté le long de l’axe Y (“Gross Domestic Product”). L’affichage est fluide, lisible et compatible avec les exigences de FreeCodeCamp.
